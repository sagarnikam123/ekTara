    Description

    • First Marathi OS interface (Prototype) on top of Windows using Java
    • developed during post graduation program, M.Sc.-Bioinformatics (Semester-III) at Bioinformatics Center, University of Pune, Pune (India)

    Usage

    • Run executable ekTar.jar
    • password - ektara

    Version

    0.1.0 - beta (retired)

    NOTE: due to presence of similar concepts from CDAC's BOSS & JNode, stops nurturing.

    Now the project is in retired state. Further, New ideas & applicable concepts are always welcome.

    Pre-requisite

    • Windows OS (7/XP)
    • JDK - 6/7
    • M-Lekhani.TTF (marathi-lekhani font)
      • from same github repository or link 1 , link 2

    • Name of OS

      • Ek Tara - The name is taken from an inspirational song sung by classmate Madhura Mohole in practical lab, "Iktara" from movie "Wake up Sid !"

    • OS concept

      • The concept of making OS in pure Java was inspired by the lectures of Priyabrata Panigrahi sir (PhD scholar at NCL), who taught us Java and inspirational story told by them about how we can use programming for solving real life problems & also for fun.
